# Peliculas-Disney
Aplicacion Backend peliculas-Disney, basado inicialmente para un challenge de Alkemy. Las tecnologías usadas son: Java, Spring Boot, PostgreSQL, JPA, y Hibernate, JUnit, Mockito.

# Stack Tecnológico
- Java 8
- Spring Boot
- Maven
- PostgreSQL
- Spring Data
- Spring Security
- JWT
- Swagger
- JUnit
- Mockito
- Envio de Emails
